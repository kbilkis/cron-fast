# Benchmark

cron-fast has been benchmarked across multiple JavaScript runtimes to ensure consistent performance.

## Runtime-Specific Benchmarks

Choose your runtime to see detailed performance comparisons:

- **[Node.js Benchmarks](benchmark-comparison-node.md)** - Tested on Node.js
- **[Deno Benchmarks](benchmark-comparison-deno.md)** - Tested on Deno

```bash
pnpm bench:bun
```
